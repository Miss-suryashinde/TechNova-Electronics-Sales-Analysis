# TechNova-Electronics-Sales-Analysis
Interactive Excel Dashboard analyzing TechNova Electronics sales data to identify factors contributing to a 12% revenue decline.

📊 TechNova Electronics Sales Dashboard

Project Overview:

This project analyzes sales performance for TechNova Electronics using Microsoft Excel. The objective was to identify possible reasons behind a 12% decline in revenue by analyzing sales across products, regions, campaigns, and months.

Business Problem:

Revenue has declined by 12%. Analyze the sales data to identify key drivers and provide recommendations to improve business performance.

Tools Used: Microsoft Excel, Pivot Tables, Pivot Charts, Dashboard, KPI Cards, Slicers, Dashboard Features, Interactive Slicers

Key Insights

1) South region generated the highest revenue.
2) East region recorded the lowest revenue.
3) Laptops were the most profitable product.
4) Google Ads generated the highest campaign revenue.
5) May recorded the highest monthly sales.
   
Recommendations

1) Improve sales performance in the East region.
2) Increase investment in Google Ads campaigns.
3) Promote high-profit products such as Smartphones.
4) Analyze successful strategies used during May.
